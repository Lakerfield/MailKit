ImapCapabilities Enum

Capabilities supported by an IMAP server.

Syntax

[System.Flags]
public enum ImapCapabilities

Remarks

Capabilities are read as part of the response to the CAPABILITY command that is issued during the connection and authentication phases of the MailKit.Net.Imap.ImapClient.
