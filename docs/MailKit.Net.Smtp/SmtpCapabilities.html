SmtpCapabilities Enum

Capabilities supported by an SMTP server.

Syntax

[System.Flags]
public enum SmtpCapabilities

Remarks

Capabilities are read as part of the response to the EHLO command that is issued during the connection phase of the MailKit.Net.Smtp.SmtpClient.

Members

Member Name Description
Authentication The server supports the AUTH extension (rfc2554), allowing clients to authenticate via supported SASL mechanisms.
BinaryMime The server supports the BINARYMIME extension (rfc3030).
Chunking The server supports the CHUNKING extension (rfc3030), allowing clients to upload messages in chunks.
Dsn The server supports the DSN extension (rfc1891), allowing clients to specify which (if any) recipients they would like to receive delivery notifications for.
EightBitMime The server supports the 8BITMIME extension (rfc2821), allowing clients to send messages using the "8bit" Content-Transfer-Encoding.
EnhancedStatusCodes The server supports the ENHANCEDSTATUSCODES extension (rfc2034).
None The server does not support any additional extensions.
Pipelining The server supports the PIPELINING extension (rfc2920), allowing clients to send multiple commands at once in order to reduce round-trip latency.
Size The server supports the SIZE extension (rfc1870) and may have a maximum message size limitation (see SmtpClient.MaxSize).
StartTLS The server supports the STARTTLS extension (rfc3207), allowing clients to switch to an encrypted SSL/TLS connection after connecting.
UTF8 The server supports the SMTPUTF8 extension (rfc6531).
