Dispose Method

Releases all resource used by the MailKit.ProtocolLogger object.

Syntax

public void Dispose ()

Remarks

Call ProtocolLogger.Dispose when you are finished using the MailKit.ProtocolLogger. The ProtocolLogger.Dispose method leaves the MailKit.ProtocolLogger in an unusable state. After calling ProtocolLogger.Dispose, you must release all references to the MailKit.ProtocolLogger so the garbage collector can reclaim the memory that the MailKit.ProtocolLogger was occupying.
