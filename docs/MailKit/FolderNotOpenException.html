FolderNotOpenException Class

The exception that is thrown when a folder is not open.

Syntax

public class FolderNotOpenException : InvalidOperationException

Remarks

This exception is thrown when an operation on a folder could not be completed due to the folder being in a closed state. For example, the IMailFolder.GetMessage(UniqueId, System.Threading.CancellationToken) method will throw a MailKit.FolderNotOpenException if the folder is not current open.
