MessageSummary Class

A summary of a message.

Syntax

public class MessageSummary : IMessageSummary

Remarks

A MailKit.MessageSummary is returned by IMailFolder.Fetch(IList<MailKit.UniqueId>,MailKit.MessageSummaryItems,System.Threading.CancellationToken). The properties of the MailKit.MessageSummary that will be available depend on the MailKit.MessageSummaryItems passed to the aformentioned method.

Attachments Property

Gets the attachments.

Syntax

public IEnumerable<BodyPartBasic> Attachments { get; }

Value

The attachments.

Remarks

Traverses over the MessageSummary.Body, enumerating all of the MailKit.BodyPartBasic objects that have a Content-Disposition header set to "attachment".

In order for this to work properly, it is necessary to include MessageSummaryItems.BodyStructure when fetching summary information from a MailKit.IMailFolder.

Body Property

Gets the body structure of the message, if available.

Syntax

public BodyPart Body { get; set; }

Value

The body structure of the message.

Remarks

The body will be one of MailKit.BodyPartText, MailKit.BodyPartMessage, MailKit.BodyPartBasic, or MailKit.BodyPartMultipart.

This property will only be set if the MessageSummaryItems.BodyStructure or MessageSummaryItems.Body flag is used when fetching summary information from a MailKit.IMailFolder.

Envelope Property

Gets the envelope of the message, if available.

Syntax

public Envelope Envelope { get; set; }

Value

The envelope of the message.

Remarks

The envelope of a message contains information such as the date the message was sent, the subject of the message, the sender of the message, who the message was sent to, which message(s) the message may be in reply to, and the message id.

This property will only be set if the MessageSummaryItems.Envelope flag is passed to IMailFolder.Fetch(IList<MailKit.UniqueId>,MailKit.MessageSummaryItems,System.Threading.CancellationToken).
