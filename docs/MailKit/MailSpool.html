SupportsUids Property

Get whether or not the service supports referencing messages by UIDs.

Syntax

public abstract bool SupportsUids { get; }

Value

true if supports uids; otherwise, false.

Exceptions

Type Reason
ObjectDisposedException The MailKit.MailSpool has been disposed.
InvalidOperationException The MailKit.MailSpool is not connected.
UnauthorizedAccessException The MailKit.MailSpool is not authenticated.

Remarks

Not all servers support referencing messages by UID, so this property should be checked before using MailSpool.GetMessageUid(int, System.Threading.CancellationToken) and MailSpool.GetMessageUids(System.Threading.CancellationToken).

If the server does not support UIDs, then all methods that take UID arguments along with MailSpool.GetMessageUid(int, System.Threading.CancellationToken) and MailSpool.GetMessageUids(System.Threading.CancellationToken) will fail.
