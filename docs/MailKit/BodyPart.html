BodyPart Class

An abstract body part of a message.

Syntax

public abstract class BodyPart

Remarks

Each body part will actually be a MailKit.BodyPartBasic, MailKit.BodyPartText, MailKit.BodyPartMessage, or MailKit.BodyPartMultipart.
