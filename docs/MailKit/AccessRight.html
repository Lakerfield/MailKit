AccessRight Struct

An individual Access Right to be used with ACLs.

Syntax

public struct AccessRight : IEquatable<AccessRight>

Remarks

An individual Access Right meant to be used with MailKit.AccessControlList.

For more information on what rights are available, see https://tools.ietf.org/html/rfc4314#section-2.1
