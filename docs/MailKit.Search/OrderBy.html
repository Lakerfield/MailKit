MailKit : MailKit.Search Namespace

OrderBy Class

Specifies a sort order for search results.

Syntax

public class OrderBy

Remarks

You can combine multiple MailKit.Search.OrderBy rules to specify the sort order that MailKit.IMailFolder.Search(SearchQuery, IList<MailKit.Search.OrderBy>,System.Threading.CancellationToken) should return the results in.
