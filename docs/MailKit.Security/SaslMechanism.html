SaslMechanism Class

A SASL authentication mechanism.

Syntax

public abstract class SaslMechanism

Remarks

Authenticating via a SASL mechanism may be a multi-step process. To determine if the mechanism has completed the necessary steps to authentication, check the SaslMechanism.IsAuthenticated after each call to SaslMechanism.Challenge(string).
